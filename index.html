a kaleidoscope instrument for stills, motion, and live performance
abs — the control's position IS the value (faders, knobs). rel — each event nudges by sensitivity × range (endless encoders; button nudges glide in smoothly). rate — deflection sets speed; release to stop (sticks, throttles). Gestures work with zero setup: unmapped trackpad gestures apply to whatever the cursor is over (source panel → slice, output → canvas) and the phone's zones do the same — mapping a gesture signal takes over from the default. Hover a target for its low → high direction. Names are editable; drag ≡ to reorder; ▾ collapses a device; pad rows carry an LED swatch (tap to cycle the hardware color). Mappings write the parameter exactly like your hand — the live output follows at the transition speed and autoplay yields the field.
